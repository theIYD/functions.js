functions.js

A hub of numerous functions with various functionalities

Latest Commit:

Install

The library is available on npm for installation. The command below can be used

                    $ npm install functions.js

Basic Usage

The library can be loaded into your application via RequireJS

                    
const func = require('functions.js');

const output = func.StringFunc.HasAtleastOneUpperCaseLetter('helLoworld');
console.log(output);
//=> returns true

If you want to use the library on the client, use the version available on unpkg.com 

                    
<script type="text/javascript"
src="https://unpkg.com/functions.js/umd/FunctionsJS.min.js"></script>

<script type="text/javascript">
      console.log(FunctionsJS.AlgoFunc.BubbleSort([5,6,9,8,4,2,1]));
      //=> [1,2,4,5,6,8,9]
</script>
